# D-racula-WM
My Dracula themed DWM build and other miscellaneous configs.

###########################################

OS - Arch Linux

Window Manager - DWM

Bar - lightly customised https://github.com/joestandring/dwm-bar

Terminal - ST

Compositor - Picom (stock)

Runner - DMENU

Font - JetBrains (available in the aur)

Browser - Brave-Beta

Player - Spotify
